# Runtime Package

This package contains the public test-slice runtime logic.

## Included
- contracts
- core helpers
- monitor logic
- sample task-run audit records
- static operator viewer files
- boundary-oriented tests

## Excluded
- live external actions
- production authority
- canonical memory mutation
- private operator/user-boundary material

## Run the tests
```bash
python3 runtime/run_tests.py
```
