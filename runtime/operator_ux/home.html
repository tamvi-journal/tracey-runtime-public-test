Tracey Runtime Public Test Viewer

Static browser pages for the public test repo. No backend. No network calls. No external libraries. No execution buttons.

Viewer-only warning

This viewer displays audit and boundary state only.

It does not execute actions.

It does not approve actions.

It does not mutate memory.

It does not write to external systems.

It does not represent production authority.
