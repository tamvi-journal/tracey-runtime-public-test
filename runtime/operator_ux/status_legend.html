Status legend

passed partial failed unknown blocked non_authoritative sandbox_only needs_approval deny
  • passed: Verification succeeded; completion claim may be allowed.
  • partial: Some evidence exists, but completion must be bounded.
  • failed: Verification failed; completion claim is forbidden.
  • unknown: Truth status is unknown; completion claim is forbidden.
  • blocked: Task cannot proceed until blocker is resolved.
  • non_authoritative: Displayed note or memory is not canonical truth.
  • sandbox_only: Action allowed only inside bounded sandbox scope.
  • needs_approval: Human approval required before action.
  • deny: Action must not proceed.
