# Contributing

## Good issues
Open an issue when you can point to one of these:
- failing test command
- unclear README or architecture doc
- boundary mismatch between docs and runtime/tests
- confusing viewer output

## Please include
- operating system
- Python version
- command you ran
- full error text
- screenshot if the viewer is the problem

## Scope
This repo is a public test harness. Keep feedback scoped to:
- monitor / gate / verification logic
- evidence handling
- viewer-only audit UX
- portability / clarity
