# Limitations

This repository intentionally stays small.

## Deliberate limits
- no live tool execution layer
- no external API calls
- no approval buttons
- no autonomous action surface
- no canonical memory authority
- no private operator identity or user-boundary files

## Public-slice constraint
This repo is a **portable test slice**, not the entire local Tracey workspace.
Some concepts from the larger workspace are intentionally omitted to keep public review focused and reproducible.
